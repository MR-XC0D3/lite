## <b>installation</b>

```
$ termux-setup-storage

$ rm -rf lite
$ pkg update
$ pkg upgrade
$ pkg install python
$ pkg install python2
$ pip install requests
$ pip install mechanize
$ pip install lolcat
$ pip install rich
$ pip install bs4
$ pkg install git
$ rm -rf lite
$ git clone https://github.com/MR-XC0D3/lite.git
$ cd lite
$ python lite.py

```
